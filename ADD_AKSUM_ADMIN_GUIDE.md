## 🔍 Troubleshooting

### If you don't see Aksum after adding:
1. Refresh the admin page (F5)
2. Check if you see any error message
3. Verify all required fields are filled
4. Make sure you're logged in as admin

### If images don't show:
- The system will use placeholder images automatically
- This is normal and ensures the site looks professional

### If admin access fails:
1. Go to: https://ethiopia-361-bg5k5.sevalla.app/login
2. Use your admin credentials
3. If you forgot credentials, check your database or contact support
