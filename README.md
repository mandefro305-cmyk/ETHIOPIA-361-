# Ethiopia 361° Tourism Website

A modern, responsive tourism website showcasing the beauty of Ethiopia with multi-language support, video backgrounds, and AI-powered chat assistance.

## 🌟 Features

- 📱 **Fully Responsive Design** - Works perfectly on all devices
- 🎬 **Video Background** - Stunning video hero section
- 🖼️ **Photo Galleries** - Multi-image galleries with lightbox
- 🤖 **AI Chat Assistant** - Intelligent tourism information
- 🌍 **Multi-Language** - English and Amharic support
- 📱 **Mobile Menu** - Hamburger menu with row-style dropdown
- 🎨 **Modern UI** - Clean, professional design

## 🚀 Live Demo

**Deployed on Vercel:** [https://ethiopia-361-tourism.vercel.app](https://ethiopia-361-tourism.vercel.app)

## 🛠️ Technology Stack

- **Backend:** Node.js, Express, EJS
- **Database:** SQLite (in-memory on Vercel)
- **Frontend:** HTML5, CSS3, JavaScript
- **File Uploads:** Multer
- **Authentication:** bcrypt, express-session
- **Deployment:** Vercel

## 📋 Installation & Setup

### Local Development
```bash
# Clone the repository
git clone https://github.com/yourusername/ethiopia-tourism-website.git
cd ethiopia-tourism-website

# Install dependencies
npm install

# Start the server
npm start
```

### Vercel Deployment
1. Push to GitHub
2. Import to Vercel
3. Automatic deployment

## 🔧 Configuration

### Environment Variables
- `NODE_ENV`: Set to 'production' for deployment
- `VERCEL`: Automatically set by Vercel

### Admin Access
- Username: `admin`
- Password: `admin`
